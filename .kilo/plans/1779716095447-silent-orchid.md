# Plan: Fix Missing Image Files Causing Build Failure

## Root Cause

The latest commit `3725984` ("footer change") deleted 3 image files from `src/images/drone-bts/` but **did not remove the corresponding gallery markdown files** that reference them:

| Deleted Image | Orphaned Markdown |
|---|---|
| `src/images/drone-bts/drone-bts.jpg` | `src/content/gallery/drone-bts.md` |
| `src/images/drone-bts/bts-image-029.jpg` | `src/content/gallery/bts-image-029.md` |
| `src/images/drone-bts/bts-image-014.jpg` | `src/content/gallery/bts-image-014.md` |

Confirmed via `git diff HEAD~1 HEAD --stat`:
```
 src/images/drone-bts/bts-image-014.jpg | Bin 675507 -> 0 bytes
 src/images/drone-bts/bts-image-029.jpg | Bin 683977 -> 0 bytes
 src/images/drone-bts/drone-bts.jpg     | Bin 3174164 -> 0 bytes
```

The images were deleted intentionally (same pattern as a prior commit `89214b5` which properly removed both images and their `.md` files). This commit missed that step.

## Fix: Delete the 3 Orphaned Gallery Markdown Files

Remove the markdown files whose images no longer exist:

1. Delete `src/content/gallery/drone-bts.md`
2. Delete `src/content/gallery/bts-image-029.md`
3. Delete `src/content/gallery/bts-image-014.md`

This matches the pattern established in commit `89214b5` ("removed a couple images from the BTS Gallery") which correctly deleted both the images and their corresponding `.md` files together.
